# Todo Tabs
#### A simple Todo list created using NextJS, ReactJS & TypeScript.

## Installation & Usage
### Installation
- cd into `Todo-Tabs`
- run: `npm i`

### Usage
- cd into `Todo-Tabs`
- run: `npm run dev`
